# Requirements: Cross-language Golden Path

> **目标**：提供 Java + curl 的标准接入路径，验证非 Python 客户端的可用性与可维护性。  
> **优先级**：P1  
> **预估工作量**：4-6 天

## 1. 功能需求

### FR-1 Java 核心路径
- [ ] 触发 Agent
- [ ] 查询状态与结果
- [ ] 错误处理、重试、超时、幂等

### FR-2 curl 对照路径
- [ ] 提供与 Java 同场景的 curl 脚本

### FR-3 文档与验收
- [ ] 输出可复制文档与可执行验证脚本

## 2. DoD

- [ ] Java 和 curl 都跑通 3 个核心场景
- [ ] 验收脚本与日志证据可复现
- [ ] 文档接入 protocol 索引
