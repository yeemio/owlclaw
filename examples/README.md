# OwlClaw Examples

示例应用，展示如何用 OwlClaw 让成熟业务系统获得 AI 自主能力。

## 示例列表

| 示例 | 场景 | 复杂度 |
|------|------|--------|
| `mionyee-trading/` | 股票交易系统接入 OwlClaw | 高 |
| `simple-cron/` | 简单 cron 任务替代 | 低 |
| `webhook-ticketing/` | Webhook 驱动的工单处理 | 中 |

## 快速开始

```bash
cd examples/simple-cron
poetry run python main.py
```
