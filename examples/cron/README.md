# Cron Trigger Examples

Examples in this folder focus on `@app.cron` usage patterns:

- `hourly_inventory.py`: hourly inventory check
- `weekday_morning_decision.py`: trading-day morning decision
- `daily_sales_report.py`: daily report generation
- `nightly_data_cleanup.py`: nightly cleanup task
- `governance_constraints.py`: governance and retry defaults
